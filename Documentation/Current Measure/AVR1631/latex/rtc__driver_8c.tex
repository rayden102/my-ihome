\subsection{Detailed Description}
X\-M\-E\-G\-A R\-T\-C driver source file. \begin{DoxyVerb} This file contains the function implementations the XMEGA RTC driver.

 The driver is not intended for size and/or speed critical code, since
 most functions are just a few lines of code, and the function call
 overhead would decrease code performance. The driver is intended for
 rapid prototyping and documentation purposes for getting started with
 the XMEGA RTC module.

 For size and/or speed critical code, it is recommended to copy the
 function contents directly into your application instead of making
 a function call.
\end{DoxyVerb}

This function initializes the R\-T\-C with period, initial count, compare value and clock prescaler. 

All the synchronized registers are written at the same time to save time. The program code need to check if the R\-T\-C is busy before calling this function. The \hyperlink{rtc__driver_8h_af9af924e641fd7a53ff85fb50a6a25ae}{R\-T\-C\-\_\-\-Busy()} function can be used to do this.
